Component Details

Every technology, its version, and key configuration

Orchestration
LangGraph StateGraph
Manages 7-node agent graph with conditional branching, parallel fan-out, and interrupt support. AsyncPostgresSaver checkpointer persists ClaimState after every node.
LLM · Reasoning
Qwen3-235B-A22B-2507
Primary model for compliance, fraud, and advisor agents via OpenRouter. 262k token context window. Fallback: Gemini 2.0 Flash Lite on 402 error.
VLM · Vision
Gemini 2.0 Flash
Receipt image → structured JSON fields (merchant, date, amount, currency, category) with per-field confidence scores. Base64-encoded; no separate OCR pipeline.
API Gateway
OpenRouter
Single endpoint for both LLM + VLM. Enables runtime model switching, 402 fallback, and cost monitoring. Accessed via OpenAI-compatible SDK.
Web Framework
FastAPI + Jinja2
Server-rendered HTML templates. Server-Sent Events for real-time agent streaming. HTMX for partial page updates. Alpine.js for lightweight interactivity.
MCP Protocol
FastMCP (Streamable HTTP)
4 MCP servers exposing tools to agents. MCP spec 2025-03-26. Health check: 406 on bare GET. mcp-rag on EC2; mcp-db, mcp-currency, mcp-email on Lambda.
Database
PostgreSQL 16
Claims, receipts, audit_log, users, eval_runs, eval_judgments, user_quota_usage. Async psycopg3 driver. 12 Alembic migrations (013 head).
Vector Store
Qdrant
expense_policies collection. ~35 chunks from 5 policy Markdown files. 384-dim cosine similarity. Embedded with all-MiniLM-L6-v2 (sentence-transformers).
ORM
SQLAlchemy + Alembic
Async mapped ORM (CamelCase Python attributes → snake_case DB columns). All schema changes via Alembic migrations. 013 revision = current head.
Compute
EC2 t3.medium
Amazon Linux 2023. Hosts FastAPI app, Qdrant, MCP-RAG, Seq. Docker Compose. 30 GB NVMe. EventBridge nightly reboot 04:00 SGT.
Serverless
AWS Lambda ARM64
mcp-db, mcp-currency, mcp-email deployed as ARM64 Lambda functions with Function URLs. Offloads stateless MCP processing from EC2.
Storage
RDS PostgreSQL 16
db.t3.micro · 20 GB SSD · private subnet. Claims database, LangGraph checkpointer, audit log. Accessible only from EC2 security group.
Input Defence
8-Layer AbuseGuard (Spec A)
B1 rate+length · B2 coherence gate · B3 injection firewall · B4 cross-check LLM · B5 hard monetary caps · B6 self-critique verifier · B7 audit trail · B8 per-user quotas.
Evaluation
DeepEval + /llmasjudge
20 benchmarks across Classification, Extraction, Reasoning, Safety, Workflow. GEval + HallucinationMetric. Self-consistency (3 runs), cross-modal, verifier-judge disagreement analysis.
Ops
EventBridge + SSM + mmga CLI
Nightly reboot via EventBridge cron. SSH-free ops via SSM Run Command. Custom ./mmga CLI for deploy, log tailing, env management, and health checks.
